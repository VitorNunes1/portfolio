## 🚀 Como Usar

### Opção 1: Abrir Localmente
1. Clone ou baixe este repositório
2. Abra o arquivo `index.html` em seu navegador
3. Pronto! O portfólio está funcionando

### Opção 2: Deploy em Servidor
1. Coloque os arquivos em seu servidor web
2. Acesse via URL do seu domínio

### Opção 3: GitHub Pages
1. Faça upload para um repositório GitHub
2. Ative GitHub Pages nas configurações
3. Seu portfólio estará live!

## 🎯 Personalização

### Alterar Cores
Edite as variáveis CSS em `styles.css`:

```css
:root {
    --primary-color: #6c5ce7;      /* Cor principal */
    --secondary-color: #a29bfe;     /* Cor secundária */
    --accent-color: #00b894;        /* Cor de destaque */
    /* ... */
}
```

### Adicionar Conteúdo
Simplesmente edite o arquivo `index.html` e atualize:
- Informações pessoais
- Experiência profissional
- Habilidades
- Links sociais

### Adicionar Imagens
1. Crie uma pasta `assets/images/`
2. Adicione suas imagens
3. Referencie no HTML: `<img src="assets/images/nome.jpg" alt="descrição">`

## 🔧 Dependências

**Nenhuma!** Este projeto usa apenas:
- HTML5 puro
- CSS3 puro
- JavaScript Vanilla

```html

```
