2. Procure por `:root {` no início do arquivo
3. Mude essas cores:

## 🌐 Colocar Online em 5 Minutos

### Opção 1: GitHub Pages (Recomendado)
1. Crie conta em github.com
2. Crie um repositório chamado `seu-usuario.github.io`
3. Faça upload dos arquivos
4. Sua página estará em: `https://seu-usuario.github.io`

### Opção 2: Vercel
1. Acesse vercel.com
2. Importe seu repositório GitHub
3. Pronto! Seu site está online

### Opção 3: Netlify
1. Acesse netlify.com
2. Arraste sua pasta para o site
3. Seu URL será gerado automaticamente

👉 Ver instruções detalhadas em `DEPLOYMENT.md`

## 🐛 Dúvidas Frequentes

### P: Como adiciono uma imagem de perfil?
**R:** Crie pasta `assets/images/`, coloque a imagem lá, e reference no HTML

### P: Como adiciono novos idiomas de programação?
**R:** Procure por `<!-- Skills Section -->` no HTML e adicione novos items

### P: Como mudo o layout/design?
**R:** Edite `styles.css` - é só CSS, não vai quebrar nada!

### P: Como coloco no meu domínio personalizado?
**R:** Ver em `DEPLOYMENT.md` → seção "Domínio Personalizado"

### P: O site carrega lentamente?
**R:** Comprima suas imagens em tinypng.com

### P: Quero adicionar mais seções?
**R:** Copie a estrutura de uma seção existente e adapte
